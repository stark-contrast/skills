# Skills
Skills are instructions to help improve performance on specialized tasks. Stark's set of skills are meant to aid users in making their web sites and apps accessible. Whether you're a developer, designer, or PM, these skills have you covered. While not required, you can pair these with [Stark's MCP server](https://www.getstark.co/support/getting-started/using-the-mcp-server/#main) for even more accessibility superpowers.

Our skills will automatically "engage" based on certain keywords, but you can force their usage by calling them with a forward slash command (ex: `/stark-pm').
